# 個人攝影集

## 概述

為完整的上線專案，前端使用 React 框架建置，後端則使用 Node.js 環境搭配 Express 框架，串接 MongoDB 以完成前端非同步請求，並實作後台系統用於上傳照片檔案。

目前網站部署在 Heroku 的服務上 → [傳送門](https://frame-of-the-state.herokuapp.com/)

## 前端實作功能

．Single Page Application
．Ajax request
．Image lazy loading

## 應用框架與套件

### 前端

．Webpack  
．React  
．React Router DOM  
．Redux  
．Flow type  
．Sass  
．Axios

### 後端

．Express  
．Mongoose  
．Handlebar  
．Multer
